# Change Log

All notable changes to the "pwl-chat" extension will be documented in this file.

## [0.0.5] - 2021-12-17
- 加入消息菜单，可以`@`用户，回复消息，收藏表情，撤回消息等；
- 加入表情包功能；

## [0.0.4] - 2021-12-16
- 加入图片上传功能

## [0.0.3] - 2021-12-15
- 紧急维护，修改活跃度更新频次，提升服务器稳定性。

## [0.0.2] - 2021-12-15
- 加入积分红包领取功能。
- 加入`@`和内置表情提示。

## [0.0.1] - 2021-12-14
- 完成初版。
